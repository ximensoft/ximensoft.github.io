植物大战僵尸 Mac版（支持M1/2/3） 详细安装教程

小新软件2024-05-012024-05-02
植物大战僵尸-游戏介绍

植物大战僵尸 for Mac是一款非常好玩的益智类策略游戏，该游戏共提供了49种功能各异的植物参加战斗，同时每个场景最多只能选用10种植物参战， 可怕的僵尸即将入侵你的家，唯一的防御方式就是你栽种的植物。赶快行动起来，挑选与你并肩奋斗的植物，一起对抗僵尸吧！

一、安装游戏

1、双击下载到Mac电脑上的 PvZ.dmg 安装包。

2、将左侧【Plants Vs.Zombies】拖入右侧【Applicatios】应用程序中进行安装，即可使用。如下所示图：

查看操作示意图

3、安装完毕，启动游戏。

友情提醒：游戏无法正常启动请参考本教程章节二、游戏无法启动问题解决

二、游戏无法启动问题解决

问题1、提示“打不开“XXX”，因为它来自身份不明的开发者。”

解决方法：打开【系统偏好设置】——【安全与隐私】——【通用】，选择最底下的【仍然打开】就可以了！

问题2：提示“XXX已损坏，无法打开。您应该将它移到废纸篓”

解决办法：

（1）打开mac电脑终端 找不到Mac终端？

（2）在终端中输入如下命令

​ sudo xattr -r -d com.apple.quarantine /Applications/PvZ.app

（3）回车后会出现【钥匙形】的图标，这时在终端内输入自己的“电脑开机密码”并回车！(终端内输入密码的时候是不会显示密码的，实际是输上了)。最后再次打开游戏即正常！注意你的电脑必须设置有密码，不可以空密码，空密码回车后会显示Sorry，try again！

查看操作示意图

注意1：你输入的命令与 sudo xattr -r -d com.apple.quarantine /Applications/PvZ.app 一个字母、一个空格都不能差，请仔细核对。最好在Mac浏览器打开本教程，然后复制这一段代码到终端中去执行。

注意2：这里如果提示 No such file:/Applications/PvZ.app 说明你没有安装 或者 安装的不是网盘里下载的PvZ.dmg安装包，请先执行安装。

问题3：“XXX已损坏，无法打开。您应该推出磁盘映像”

（1）打开mac电脑终端 找不到Mac终端？

（2）在终端中输入如下命令，点击回车

sudo sudo spctl --master-disable

（3）回车后会出现【钥匙形】的图标，这时在终端内输入自己的“电脑开机密码”并回车！(终端内输入密码的时候是不会显示密码的，实际是输上了)。最后再次打开游戏即正常！注意你的电脑必须设置有密码，不可以空密码，空密码回车后会显示Sorry，try again！

注意：你输入的命令与 sudo spctl --master-disable 一个字母、一个空格都不能差，请仔细核对。最好在Mac浏览器打开本教程中，然后复制这一段代码到终端中去执行。
